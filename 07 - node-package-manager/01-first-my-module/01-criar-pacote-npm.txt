  * npm version  atualiza a versão do package (1.2.0 = MAINGER.MAINER.PATCH)
    - MAIJOR (Major) => NOVA VERSÃO DE SOFTWARE
    - MAINER (Minor) => FEATURE, NOVA FUNCIONALIDADE
    - PATCH (Path) => ALTERAÇÃO DE BUG NO SISTEMA

  8º npm version patch
  9º rm -rf .git
  10º npm publish

  11º npm version major
  12º npm publish
